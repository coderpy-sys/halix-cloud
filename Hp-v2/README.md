# Halix Cloud (panel)

Halix Cloud **Hp-v2** is a Proxmox-focused hosting panel rebranded for Halix with a deep purple theme and Halix Cloud UI styling.

- **Install:** `install/README.md` and `install/halix-cloud-install.sh` (default clone: `https://github.com/coderpy-sys/halix-cloud.git`)
