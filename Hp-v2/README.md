# Halix Cloud (panel)

Halix Cloud **Hp-v2** is a Proxmox-focused hosting panel based on [Convoy](https://github.com/ConvoyPanel/panel), rebranded for Halix with a purple UI theme, Coterm **admin** routes removed, and optional **PCT (LXC)** work tracked in `docs/PCT.md`.

- **Install:** `install/README.md` and `install/halix-cloud-install.sh` (default clone: `https://github.com/coderpy-sys/halix-cloud.git`)
- **Upstream docs:** [docs.convoypanel.com](https://docs.convoypanel.com)
- **License:** See `LICENSE.md` and `LICENSE` in this directory.
